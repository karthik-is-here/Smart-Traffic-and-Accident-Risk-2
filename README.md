# Smart-Traffic-and-Accident-Risk-2

Urban traffic congestion and road accidents are growing challenges in rapidly developing cities like Kochi. Most existing traffic management systems operate reactively — responding only after problems occur. This project addresses that gap by building a proactive, data-driven web application that predicts traffic congestion levels and accident risk for individual roads across seven local areas in Kochi: Edappally, Kakkanad, Fort Kochi, Mattancherry, Vyttila, Palarivattom, and Kadavanthra.
The system uses XGBoost machine learning models trained on a synthetic dataset of over 50,000 records, capturing features such as road type, number of lanes, speed limits, time of day, day of the week, weather conditions, and historical traffic volume. Two separate models are trained — one for congestion level classification and one for accident risk classification — both producing predictions across four severity levels: Low, Moderate, High, and Very High.
The frontend is built with Streamlit and presents results through an interactive Folium map interface, colour-coded road markers, sortable data tables, and a live danger alert panel that flags high-risk roads under the selected conditions. Users can configure inputs through a clean dashboard sidebar and switch between congestion and risk views instantly.
Tech Stack: Python · Streamlit · XGBoost · Scikit-learn · Folium · Pandas · NumPy
